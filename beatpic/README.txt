Instructions to running Project2.
REQUIRES PYTHON3!!!!!!!!!
If you have pysynth, numpy, and flask please skip to step 8.
1)Open terminal 
2) Install flask enter: pip install flask
3) Install numpy enter: pip install numpy
4) Instal pysynth: enter: clone https://github.com/mdoege/PySynth.git
5) Once it finishes cloning enter: cd PySynth
6) In the directory enter: python3 setup.py install
7) Then enter: cd
8) Then go to the folder project2-235 then cd beatpic.
9) Run run.py enter: python3 run.py
10) Go to your browser and enter: localhost:5000
11) enjoy!
